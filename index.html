      • Ctrl+S = zapisuje plik
      • Ctrl+Z = cofa ostatnią zmianę
      • Ctrl+C = kopiuje plik lub tekst
      • Ctrl+V = wkleja plik lub tekst

      Rodzaje złącz USB

      1. USB A - Najczęściej spotykane złącze, do niedawna będące standardem.

      2. USB B - Ma nieco kwadratowy kształt. Czasami spotykane jako złącze USB do drukarki.

      3. USB C - Najnowszy typ złącza, który powoli staje się też najpopularniejszym.
